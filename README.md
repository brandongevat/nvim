# Installation
## Pre-requisites
1. [NVIM 0.9.0+](https://github.com/neovim/neovim/releases/)
2. [Packer](https://github.com/wbthomason/packer.nvim)

## Installation
1. Clone the repo:
    ```bash
    git clone https://github.com/brandongevat/nvim
    ```
2. Open nvim 
    ```bash
    nvim .
    ```

3. Sync packer
    ```bash
    :PackerSync
    ```

4. Restart nvim and enjoy!
